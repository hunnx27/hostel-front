# 개발 히스토리
## 0. 기타 환경 잡기
`npm install --global yarn`
npm보다 좀더 나음 비슷함 

## 1. create-react-app 생성
`create-react-app hostel-front`
리액트 앱을 생성한다.

## 2. 앱 실행
`yarn start`
서버가 켜짐

## 3. App.js 에서 변경

## 4. git 연결
1. 깃허브에서 레파지토리 생성후 
2. 수정된 파일들 스테이징 올리기, 커밋하기
2. 리모트와 연결
`git remote add origin https://github.com/hunnx27/hostel-front.git`
3. 깃 푸시하기
`git push --set-upstream origin master`
